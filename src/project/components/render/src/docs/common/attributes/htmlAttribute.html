htmlAttribute

htmlAttribute attribute allows developers to set custom HTML attributes on custom tag generated elements.

Examples

Custom Tag

<E2:Form htmlAttribute="{"className":"demo-form", "id":"demo-form"}">
</E2:Form>

Sample JSON

{
    "type": "form",
    "attributes": {
        "name": "form1",
        "method": "POST",
        "action": "http://www.ny.gov",
        "className": "demo-form",
        "id": "demo-form"
    },
    contents: []
}

For the purposes of this example we are using the form. For a full support list see below. In this particular example, the E2:Form custom tag only renders a form HTML element. So for this example the htmlAttribute will work fine. Other custom tags require more specific attributes. The support list below explains which variation of the attribute should be used.

Rendered Example

In this example the form tag has been extended to include the id and class attributes. To see the rendered results use the browsers development tool to inspect the box below.

This attribute can be set on the following custom tags:

Standard htmlAttributes
  • Button - Button
  • ButtonGroup - Div (ButtonRow), Div(ButtonCol)
  • CheckBoxGroup - Fieldset
  • Col - Div
  • From - Form
  • Hidden - Input
  • InputGroup - Fieldset
  • Message - Div
  • Notifier - Span or A (element depends on render mode)
  • Notifier Group - UL
  • RadioGroup - Fieldset
  • Row - Div
  • Section - Section (Section), Header - (SectionHeader), Div - (SectionCol), Div - (SectionIndicator)

    • Element Specific htmlAttributes

    • Checkbox
      • label - labelHtmlAttribute
      • input - inputHtmlAttribute
    • Date
      • label - labelHtmlAttribute
      • input - inputHtmlAttribute
    • SelectBox
      • label - labelHtmlAttribute
      • input - inputHtmlAttribute
    • Radio
      • label - labelHtmlAttribute
      • input - inputHtmlAttribute
    • Textarea
      • label - labelHtmlAttribute
      • textarea - inputHtmlAttribute
    • Text
      • label - labelHtmlAttribute
      • input - inputHtmlAttribute
