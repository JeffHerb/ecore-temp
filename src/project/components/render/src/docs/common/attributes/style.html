Style Attribute

In Review

The style attribute is used to define special reusable styles across multiple components. Multiple styles can be defined by adding the attribute to a supported component and comma seperated the different style values found below. When this attribute is applied to an element it will appear as the style property of the root object it is applied too. The style value is based

  • align-right
  • min-width
  • negative-number

Align Right

Align right switches the text alignment from the default left to right.

Example JSP Tag

<E2:Text viewKey="text1" style="align-right" />

Example JSON

{
    "type": "text",
    "template": "field",
    "label": {
        "text": "Demo Text Label",
        "attributes": {
            "for": "demo"
        }
    },
    "input": {
        "attributes": {
            "name": "demo",
            "id": "demo",
            "type": "text",
            "value": "Example Text"
        },
        "style": "align-right"
    }
}

Rendered Examples

This attribute can be set on the following custom tags:

  • TableColumn
  • Text

Mid Width

Min Width attribute forces columns to take up as little space as possible.

Example JSP Tag

<E2:TableColumn label="Index" name="index-col" hideLabel="true" style="mid-width" />

Example JSON

{
    "template": "table",
    "head": {
        "rows": [
            {
                "columns": [
                    {
                        "type": "header",
                        "text": "Index",
                        "hideLabel": true,
                        "style": "min-width"
                    },
                    {
                        "type": "header",
                        "text": "Full Name"
                    },
                ]
            }
        ]
    },
    "body": {
        ... Omitted Body Rows ...
    }
}

Rendered Examples

  • TableColumn
