PageTitle Custom Tag

This is the common page title that should appear on the top over every Empire page.

Examples

Custom Tag

<E2:PageTitle title="Page Name" subTitle="Page Sub Title">
</E2:PageTitle>

Sample JSON

{
    "template": "pageTitle",
    "title": "Page Name",
    "subtitle": "Page Sub Title"
}

PageTitle is very limited in option, developer simply need to to supply the page title, and a subtitle if needed.

Default example

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
title String No General page title
subtitle String No Specific page sub-title
visibility String No Used to control the visibility state of a particular input and/or its label or parts (more details) Unlike the older tag developer can no longer disable page level buttons.
Acceptable Options:
  • HIDE_ON_READ
  • HIDE_ON_UPDATE
