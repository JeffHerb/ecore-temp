Output Tag

The <E2:Ouput> is for output static content to the screen.

Examples

Custom Tag

<E2:Output text="<%= output %>">

Sample JSON

{
    "template": "output",
    "text": "<p>This is an example of the output tag placing raw html onto the page.</p>"
}

<E2:Output> tag is used to display predefined static content that is not part of the Registration tool.

Rendered Examples

It is important to note, all static output that is kicked out from the E2:Output tag is pre wrapped in a div with the class of emp-ouput. All htmlAttribute related items will also be appended to this wrapper div.

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
text String Yes The viewKey that has been registered in the Framework Registration Tool.
htmlAttributes String No Adds HTML attributes to the rendered wrapper <div> container (more details)
printOnly Boolean No Indicates this section should visible only when the page is being printed.

Print Only Attribute

Custom Tag

<E2:Output text="<%=  %>" printOnly="true">

Sample JSON

{
    "template": "output",
    "text": "<p>This is only </p>",
    "printOnly": true

}

<E2:Output> with the printOnly attributes simply dont render visually on screen but appear when printed.
