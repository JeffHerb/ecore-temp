Row Custom Tag

The row tag is used to define the occurrence of a new grid row. Grid rows by default are defind by the class: `cui-row`

Examples

Custom Tag

<E2:Row>
</E2:Row>

Sample JSON

{
    "type": "row",
    "template": "grid",
    "contents": []
}

E2:Row uses the shared grid template. This means the tag at a minimum should output JSON properties of type and template. The row tag should also create propety of contents which is an array for each sub object tag created within.

Default example

By default rows render no visible content. Use the browser development tool on the box below to see the rendered output.
