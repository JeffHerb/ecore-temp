Column Custom Tag

New in E2

The col tag defines a column within a row of a grid. Columns are used as containers for other tags and are meant to define the layout of those tags. Column widths are defined with the classes emp-col-half or emp-col-full.

Examples

Custom Tag

<E2:Col>
</E2:Col>

Sample JSON

{
    "type": "column",
    "template": "grid",
    "contents": []
}

<E2:Col> uses the shared grid template. This means at a minimum the JSP tag should output JSON properties of type and template. The column tag should also create propety of contents which should contain an array of JSON objects representing all JSP tags contained inside.

Default example

This example establishes a grid by wrapping columns inside of rows. All <E2:Col> tags must be contained inside a <E2:Row> tag.

By default <E2:Col> render no visible content. Use the browser development tool on the box below to see the rendered output.

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
width String No Defines the width of the column using the following values:
  • full - 100% of the row's width
  • half - 50% of the row's width
htmlAttributes String No Adds HTML attributes to the rendered <div> container (more details)

Using the Width Attribute

By default developers dont have to specify a column width. Any <E2:Col> tags missing the width attribute will default to using the full width of the parent <E2:Row>.

Width Attribute Example

Custom Tag

<E2:Row>

    <E2:Col width="full">
    </E2:Col>

</E2:Row>
<E2:Row>

    <E2:Col width="half">
    </E2:Col>

    <E2:Col width="half">
    </E2:Col>

</E2:Row>

Size Extended JSON

{
    "type": "row",
    "template": "grid",
    "contents": [
        {
            "type": "column",
            "template": "grid",
            "width": "full",
            contents: []
        }
    ]
},
{
    "type": "row",
    "template": "grid",
    "contents": [
        {
            "type": "column",
            "template": "grid",
            "width": "half",
            contents: []
        },
        {
            "type": "column",
            "template": "grid",
            "width": "half",
            contents: []
        }
    ]
}

Rendered Size example

For demonstration purpose this time the demo object contains 2 different <E2:Row>. Inside the first row, a child <E2:Col> should be create and is being forced to take the full width of the row because the size attribute is set to full. The second row has 2 children, each of which has the size attribute set to half.
