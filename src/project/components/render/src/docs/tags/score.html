Score Custom Tag

In Review

The <E2:Score> is used to create a visual representation of a percentage value. This component can only be used inside an E2:TableColumn.

Examples

Custom Tag

<E2:Score score="<% score %>" base="<% base %>" />

Sample JSON

{
    "template": "score",
    "color": "green",
    "score": 100,
    "percentage": 100
}

E2:Score custom tag is used when there needs to be a visual represnetation of a percentage based value inside the cell of a table. Here the developer simply need to provide a max (base) value and the related raw score value. Framework will then used the two to create the percentage.

The JSON markup for this tag is very lean. The properties score is pulled directly out of the provided developer markup. The percentage is figured out on the framework layer to avoid possible floating point issues on the front-end. Developers can also pick from a varity of bar colors. They can do this by providing the name of the color they wish the bar to appear as. The default value is set to green.

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
score Number Yes The raw score
base Number No The total maxamum amount the score is based on. This is used to calculate the percentage.
color String No The desired score the background bar should appear as. Allowed colors:
  • Green default
  • Red
  • Yellow
