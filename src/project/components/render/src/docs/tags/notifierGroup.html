NotifierGroup Tag

The <E2:NotifierGroup> is used to group notifiers together in a list.

Examples

Custom Tag

<E2:NotifierGroup>
    <E2:Notifier notifierID="note1">
    <E2:Notifier notifierID="note2">
    <E2:Notifier notifierID="note3">
</E2:NotifierGroup>

Static JSON

{
    "template": "notifierGroup",
    "notifiers": [
        ... Notifiers Omitted ...
    ]
}

When the E2:NotifierGroup is render into JSON it creates an object with two properties. The first is the template attribute, which ensures the render builds notifiers inside an unordered list. The second is the notifiers array property. The contents of this property should be an array of object which match the E2:Notifier specification.

Static Rendered Examples

The E2:NotifierGroup is a simple wrapper container. Developers are encourage to use this tag when the notifiers dont shown inside of a table.
