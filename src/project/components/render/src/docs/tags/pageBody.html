PageBody Tag

The <E2:PageBody> is used to wrap all of the page specific components in an accessible container.

Examples

Custom Tag

<E2:PageBody>

Sample JSON

{
    "template": "pagebody",
    "contents": []
}

<E2:PageBody> tag should be used to wrap all of the definitive page contents. This tag appears under the global header and includes the <E2:PageTitle> tag as well as all groups.

Rendered Examples

This tag does not render any visiual content. The tag simply class the create a main container and by default includes the role attribute, set to main. At the JSON layer, this tag simply requests for the pageBody template.

By default this renders no visible content. Use the browser development tool on the box below to see the rendered output.

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
title String Yes This is the title for the page. This value is sent to the global page header JSON under the page object and in the title property.
mode String No This is the title for the page. This value is sent to the global page header JSON under the page object and in the mode property.
htmlAttributes String No Adds HTML attributes to the rendered wrapper <main> container (more details)
