Form Custom Tag

All fields on a page are contained inside form tags.

Examples

Custom Tag

<E2:Table>
    <E2:TableControls>
    </E2:TableControls>
    <E2:TableColumn label="Column 1" />
    <E2:TableColumn label="Column 2" />
    <E2:TableColumn label="Column 3" />
</E2:Table>

Sample JSON

{
    "template": "table",
    "title": "Table Control Example",
    "attributes": {
        "id": "table_id"
    },
    "controls": {
        "contents": [
            {
                "type": "text",
                "template": "field",
                "label": {
                    "attributes": {
                        "for": "E_FILTERS"
                    },
                    "text": "Filters"
                },
                "input": {
                    "attributes": {
                        "type": "text",
                        "id": "E_FILTERS",
                        "name": "E_FILTERS",
                    }
                }
            }
        ]
    },
    ... Ommited the remaining of the table ...
}

<E2:Form> tag is a sepcial container used to place sever side table controls above a table. The tag itself accepts any other E2

When turned into JSON, the values of the E2:TableContorls tag are stored inside of the table definition under the controls property. Inside of this property is a contents array property that is manually called by the render. The contents of the individal array items should follow the custom tag JSON requirements for the custom tags provided within.
