Icon Custom Tag

This custom tag is used to add visual icon based flags.

Examples

Custom Tag

<E2:Icon icon="checkmark" title="Checked!" />

Sample JSON

{
    "template": "icon",
    "attributes": {
        "title": "Checked!"
    },
    "icon": "checkmark"
}

E2:Icon tag is used to provide developers with a way to add icons and accessible text in places were controls dont normally exist.

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
icon String Yes Name of the icon that needs to be displayed
text String No Additional hidden accessibility text for a screen reader
title String Yes Text value for this icon.
htmlAttributes String No Adds HTML attributes to the rendered <button> or <a> (link) tag (more details)
