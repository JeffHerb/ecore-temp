HiddenField Custom Tag

The <E2:HiddenField> creates hidden input fields.

Examples

Custom Tag

<E2:HiddenField elementName="E_SESSION_ID" value="12345" />

Sample JSON

{
    "type": "hidden",
    "template": "field",
    "input": {
        "attributes": {
            "name": "E_SESSION_ID",
            "id": "E_SESSION_ID",
            "type": "hidden",
            "value": "12345"
        }
    }
}

Hidden fields can be placed anywhere inside a <E2:Form> tag. The only required attribute is name.

Default example

Hidden inputs can be placed anywhere inside of a form tag and do not render any visible content. Use the browser's development tool to inspect the box below to see the rendered output of the example JSON above.

Options

Custom Tag Attributes
Attribute Name Type Required Purpose
name String Yes Name of the hidden input
value String No Initial value the input
rsMapping String No Maps the field to a result set. Please see the ResultSet custom tag documentation for details.
htmlAttributes String No Adds HTML attributes to the rendered <input> element (more details)
